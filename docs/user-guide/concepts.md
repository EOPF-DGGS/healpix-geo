# Fundamental Concepts

## Overview

HEALPix is a spherical tiling system with three fundamental properties:

1. **Equal Area** - All cells have the same area
2. **Iso-latitude** - Cells are arranged in rings of latitude
3. **Hierarchical** - Natural tree structure
